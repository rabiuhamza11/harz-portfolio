Payhip Product Catalog — Harz Digital Services

12 products ready for manual upload to Payhip | All download files hosted on GitHub Pages

📋 How to add each product on Payhip:

  1. Go to payhip.com → Products → Add New Product → Digital Product
  2. Copy the Product Name from below and paste it
  3. Copy the Description and paste it
  4. Set the Price as shown
  5. For the Download File, paste the GitHub Pages URL (Payhip will fetch it)
  6. For Cover Image, paste the cover image URL
  7. Click Publish — done!

Note: Payhip supports Stripe and PayPal for payments. Connect your Wise UK account for Stripe, or use Payhip's native checkout.
