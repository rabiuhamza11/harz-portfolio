Description
Send messages to any of 20 AI agents powered by llama-3.3-70b. Returns AI responses with usage tracking.

action
string
"chat", "agents", "info", "generate_key"
required

string

agent_id
string

string
API key for auth
Example Request

Harz JavaScript SDK
Drop-in SDK for web and Node.js. No dependencies.
Installation
Browser: <script src="https://rabiuhamza11.github.io/harz-portfolio/harz-sdk.js"></script> Node.js: npm install @harz/sdk
Quick Start
// Initialize const harz = new HarzSDK({ apiKey: 'hk_your_api_key', defaultAgent: 'agent_01' }); // Chat with AI const reply = await harz.cloud.chat({ message: 'What products do you recommend?', agentId: 'agent_03' }); console.log(reply.response); // List agents const agents = await harz.cloud.agents(); // Generate API key const key = await harz.cloud.generateKey('user@email.com');
Product Catalog
// Browse products const products = await harz.catalog.list({ limit: 20 }); // Search const results = await harz.catalog.search('engineering'); // Get details const product = await harz.catalog.get('product_id'); // Filter by category const music = await harz.catalog.byCategory('music');
Payments
// Create checkout const checkout = await harz.pay.checkout({ amount: 1299, // $12.99 in cents currency: 'USD', email: 'customer@email.com', productId: 'prod_123' }); // Redirect to payment window.location.href = checkout.authorization_url; // Verify payment const status = await harz.pay.verify(checkout.reference); // GDEG token payment const gdeg = await harz.pay.gdeg({ amount: 2610, wallet: '0x0B63...3C1' });
CRM
// Log inquiry await harz.crm.logCustomer({ phone: '2348012345678', name: 'John Doe', message: 'I want to buy the Notion bundle' }); // Auto-reply const reply = await harz.crm.autoReply({ message: 'How much is the engineering collection?', phone: '2348012345678' }); // Stats const stats = await harz.crm.stats();
Phone Lookup
const info = await harz.lookup.phone('08028687857'); console.log(info.network); // "MTN" console.log(info.region); // "Lagos" console.log(info.type); // "GSM"
