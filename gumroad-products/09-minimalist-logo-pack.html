📐 How to Customize

  1. Open SVG file in any vector editor (Illustrator, Inkscape, Figma, or even text editor)
  2. Change stroke color by modifying the stroke attribute
  3. Adjust stroke-width for thinner/bolder lines
  4. Add your business name below the icon
  5. Export as PNG/SVG for web or print
