About JWT

A JWT has 3 parts separated by dots: header.payload.signature 1. Header — algorithm and token type 2. Payload — claims (user data, expiry, etc.) 3. Signature — verifies token integrity This tool only decodes. It does NOT verify signatures. Never paste production secrets.
